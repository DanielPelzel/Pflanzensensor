# Pflanzensensor 

**Problem** Pflanzen vertrocknen regelmäßig

**Lösung** Ein Sensor der die Bodenfeuchtigkeit misst. Über einen Microcontroller werden diese Daten eingelesen. Fällt die Feuchtigkeit unter einen gewissen Wert, geht eine LED an

## Hardware
Microcontroller Arduino R3 Uno  
220Ohm Resistor   
LED Rot   
Kapazitiver Bodenfeuchtigkeitssensor (Korrosionsbeständig!!)  

## Pin Belegung 
Sensor Data - A0
Sensor plus - 5V
Diode - 13
